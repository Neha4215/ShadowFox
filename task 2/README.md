# Basic Calculator App

A simple Android calculator application with basic arithmetic operations.

## Features
- Addition, subtraction, multiplication, and division
- Clear button to reset the calculation
- Decimal point support
- Error handling for division by zero
- Intuitive user interface

## Technical Details
- Developed in Java for Android
- XML layout files for UI design
- Supports Android SDK 21 and above

## How to Build
1. Open the project in Android Studio
2. Sync the project with Gradle files
3. Build and run the application

## Usage
1. Tap the number buttons to enter digits
2. Tap an operator button (+, -, *, /) to select an operation
3. Tap another number button to enter the second operand
4. Tap the equals (=) button to see the result
5. Tap the clear (C) button to reset the calculator
